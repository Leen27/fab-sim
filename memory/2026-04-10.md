# 2026-04-10 - 半导体工厂仿真项目启动

## 今日开局
队长抛出了一个超大项目：**用 Godot 制作半导体工厂仿真应用**
- 核心功能：工厂 3D/2D 可视化仿真
- 关键能力：与真实 MES 系统交互
- 终极目标：用仿真环境验证 MES 系统逻辑
- 起手式：先设计工厂

## 项目分析
这是一个典型的**数字孪生 (Digital Twin)** 应用场景：
1. 虚拟工厂 ←→ 真实 MES 数据流
2. 仿真验证生产逻辑、排程算法
3. 可视化展示产线状态、设备运转

## 使用方法

### 方式一：本地开发
```bash
# 启动 MES 模拟服务器
./start.sh mes

# 启动 Godot 编辑器 (本地版)
./start.sh godot
```

### 方式二：外网访问 (推荐)
```bash
# 创建外网访问隧道
./start.sh tunnel

# 然后在浏览器中打开：
# https://editor.godotengine.org/release/4.3-stable/
# 导入项目地址：
# https://organ-gem-cannon-slideshow.trycloudflare.com/project.godot
```
