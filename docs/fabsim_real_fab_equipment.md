# 真实Fab设备配置差异

## 一、不是所有Fab都需要全部设备

### 核心四大件（任何Fab都有）

```
┌─────────────────────────────────────────────────────────┐
│                  必需设备（100%有）                      │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   [清洗] → [沉积] → [光刻] → [刻蚀]                    │
│    🟨       🟩       🟦       🟥                       │
│                                                         │
│   这四样是半导体制造的"四大金刚"                        │
│   没有它们就做不出芯片                                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

| 设备 | 为什么必需 |
|------|-----------|
| **清洗** | 每步前后都要洗，不做就污染 |
| **沉积** | 不沉积材料就没东西刻 |
| **光刻** | 不曝光就没有图案 |
| **刻蚀** | 不刻蚀就去不掉多余材料 |

## 二、不同Fab的设备差异

### 按工艺节点分

| 工艺节点 | 典型设备差异 |
|----------|-------------|
| **28nm及以上**（成熟制程） | 不需要EUV光刻机，DUV就够了；ALD可选 |
| **14nm/7nm/5nm**（先进制程） | 必须有EUV光刻机；ALD必需；多重曝光设备 |
| **3nm及以下** | EUV+High-NA EUV；更多ALD步骤 |

### 按产品类型分

| 产品类型 | 特殊设备需求 |
|----------|-------------|
| **逻辑芯片**（CPU/GPU） | CMP必需（多层铜互连），离子注入多 |
| **存储芯片**（DRAM/Flash） | 特殊刻蚀设备（高深宽比），CMP相对少 |
| **功率器件** | 不需要最先进光刻，退火设备多 |
| **模拟芯片** | 工艺步骤少，设备相对简单 |
| **MEMS传感器** | 需要特殊释放刻蚀，其他相对简单 |

### 按产线阶段分

| 产线类型 | 设备配置 |
|----------|----------|
| **研发线（R&D）** | 设备种类全，但每种可能只有1台 |
| **量产线（Mass Production）** | 核心设备多机台（10+台光刻机），辅助设备精简 |
| **中试线（Pilot）** | 介于两者之间 |

```

│  检测机      →   小厂可能送检外包                      │
│  退火炉      →   有些工艺用快速热退火替代                │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 四、真实案例

### 案例1：成熟制程代工厂（如部分国产Fab）

```
工艺：28nm/40nm/65nm 逻辑芯片

设备配置：
✅ 清洗机（多）
✅ 氧化炉/CVD/PVD
✅ DUV光刻机（ASML 1980Di等）
✅ 干法刻蚀机
✅ 离子注入机
✅ CMP设备（较少）
✅ 检测设备
❌ EUV光刻机（用DUV多重曝光替代）
❌ ALD（可选，看具体层次）

总设备类型：8~10种
```

### 案例2：先进制程大厂（如台积电3nm Fab）

```
工艺：3nm 逻辑芯片

设备配置：
✅ 清洗机（非常多）
✅ 氧化炉/CVD/ALD（ALD比重很高）
✅ EUV光刻机（昂贵，几十台）
✅ 干法刻蚀机（非常多）
✅ 离子注入机
✅ CMP设备（每层都要CMP）
✅ 检测设备（全自动化）
❌ 湿法刻蚀（大部分层次用干法）

总设备类型：12~15种
```

### 案例3：功率器件Fab

```
工艺：IGBT/MOSFET

设备配置：
✅ 清洗机
✅ 氧化炉（特别多）
✅ 光刻机（不需要太先进，DUV）
✅ 刻蚀机
✅ 离子注入机（特别多，不同能量）
✅ 退火炉（特别多）
❌ CMP（很少用）
❌ EUV
❌ ALD

总设备类型：6~8种
```

### 案例4：存储芯片Fab（如长鑫、三星DRAM）

```
工艺：1x/1y/1z nm DRAM

设备配置：
✅ 清洗机
✅ CVD/ALD（电容需要ALD）
✅ DUV光刻机
✅ 特殊刻蚀机（高深宽比刻蚀）
✅ 离子注入
✅ CMP
✅ 检测设备
❌ EUV（DRAM层数多但线宽要求相对宽松）

总设备类型：9~11种
```

## 五、MVP选择的依据

### 为什么MVP选这6种？

```
[清洗] → [沉积] → [光刻] → [刻蚀] → [热处理] → [检测]

覆盖范围：
✅ 任何Fab都有这些（或类似功能设备）
✅ 能完整走通一个工艺流程
✅ 能验证仿真核心逻辑
```

### 如果只做最精简版本

```
4设备版本（极限MVP）：

[清洗] → [沉积] → [光刻] → [刻蚀] → 检测外包

这个配置甚至能做出简单芯片（虽然良率可能不高）
```

## 六、给你的结论

| 问题 | 答案 |
|------|------|
| 所有Fab都需要14+种设备吗？ | **不需要**，看工艺和产品 |
| 最少需要几种？ | **4种**（清洗+沉积+光刻+刻蚀）能做出芯片 |
| 常见配置有几种？ | **8~12种**是大多数Fab的配置 |
| 仿真需要模拟全部吗？ | **不需要**，覆盖核心流程即可 |

**FabSim MVP选6种设备的原因：**
1. 覆盖95% Fab的核心流程
2. 能验证"设备-缓冲-调度"完整逻辑
3. 后期扩展容易（加分设备即可）
