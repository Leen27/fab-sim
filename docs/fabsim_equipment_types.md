# 半导体前道设备类型说明

## 一、真实Fab需要多少种设备？

### 完整设备清单（14+种）

| 步骤 | 设备类型 | 英文简称 | 作用 |
|------|----------|----------|------|
| 清洗 | 湿法清洗机 | Wet Bench | 化学药液清洗 |
| 氧化 | 氧化炉/扩散炉 | Furnace | 高温氧化 |
| 沉积 | CVD设备 | CVD | 化学气相沉积 |
| 沉积 | PVD设备 | PVD | 物理气相沉积（溅射）|
| 沉积 | ALD设备 | ALD | 原子层沉积（先进制程）|
| 光刻 | 涂胶机 | Coater | 涂光刻胶 |
| 光刻 | 光刻机 | Scanner | 紫外线曝光 |
| 光刻 | 显影机 | Developer | 显影+清洗 |
| 刻蚀 | 干法刻蚀机 | Dry Etcher | 等离子体刻蚀 |
| 刻蚀 | 湿法刻蚀机 | Wet Etcher | 化学药液刻蚀 |
| 注入 | 离子注入机 | Implanter | 离子掺杂 |
| 热处理 | 退火炉 | RTA/Annealer | 快速退火 |
| 平坦化 | CMP设备 | CMP | 化学机械抛光 |
| 检测 | CD-SEM | Metrology | 线宽测量 |
| 检测 | 缺陷检测设备 | Inspection | 颗粒/缺陷扫描 |

**结论：真实Fab有14~20种不同类型设备**

## 三、MVP产线布局示例

### 单层6步工艺流程

```
晶圆入口
    ↓
[🟨清洗] → [🟩沉积] → [🟦光刻] → [🟥刻蚀] → [🟧热处理] → [⬜检测] → 出口
    ↓         ↓          ↓          ↓           ↓
  [🟪]      [🟪]       [🟪]       [🟪]        [🟪]
  缓冲1     缓冲2      缓冲3      缓冲4       缓冲5

Recipe配置：
- 清洗: 10分钟
- 沉积: 30分钟  
- 光刻: 45分钟
- 刻蚀: 30分钟
- 热处理: 20分钟
- 检测: 15分钟

总加工时间: 150分钟/层
```

### 多层循环（Phase 2）

```
同一个设备组，Lot循环N次：

Lot #1: 清洗→沉积→光刻→刻蚀→热处理→检测 (Layer 1完成)
        ↓
       回到清洗，开始 Layer 2
        ↓
Lot #1: 清洗→沉积→光刻→刻蚀→热处理→检测 (Layer 2完成)
        ↓
       循环40次...
```

## 五、扩展路线图

### MVP（现在）：6种设备
- 清洗 + 沉积 + 光刻 + 刻蚀 + 热处理 + 检测

### Phase 2：拆分到10种
- 清洗
- 氧化炉（分离出来）
- CVD沉积
- PVD沉积  
- 涂胶机
- 光刻机
- 刻蚀机
- 离子注入
- CMP
- 检测

### Phase 3：完整14+种
- 所有真实设备类型
- 支持不同机台的不同加工能力

## 六、给你的建议

**MVP阶段只用6种设备就够了：**

1. 界面清爽 — 不会一堆图标看花眼
2. 逻辑清晰 — 每种设备作用明确  
3. 容易扩展 — 后期拆分不影响已有架构
4. 验证核心 — 证明"多设备+缓冲+调度"能跑通

**真实Fab虽然设备多，但仿真的价值在于验证"流"和"调度"，不是完美复刻每一台机器。**
