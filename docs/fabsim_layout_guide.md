# 半导体产线构成说明

## 一、产线 = 三大件

```
┌─────────────────────────────────────────────────────────┐
│                      一条产线                            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   [设备]      [设备]      [设备]      [设备]            │
│    🟦─────────🟦─────────🟦─────────🟦                  │
│   光刻机     刻蚀机     沉积机     清洗机               │
│      │         │          │         │                   │
│      └─────────┴────🟪────┴─────────┘                   │
│                  缓冲区                                  │
│                                                         │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐               │
│  │  设备    │  │  缓冲    │  │  传输    │               │
│  │ (加工)   │  │ (暂存)   │  │ (路径)   │               │
│  └──────────┘  └──────────┘  └──────────┘               │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 二、核心设备类型（Fab常用）

### 前端工艺设备

| 设备类型 | 作用 | 简写 | FabSim表示 |
|----------|------|------|------------|
| **光刻机** | 把电路图案曝光到晶圆上 | Litho | 蓝色方块 🟦 |
| **刻蚀机** | 按图案刻掉多余材料 | Etch | 红色方块 🟥 |
| **薄膜沉积** | 在晶圆上沉积薄膜 | CVD/PVD | 绿色方块 🟩 |
| **离子注入** | 改变晶圆电特性 | IMP | 紫色方块 🟪 |
| **清洗机** | 清洁晶圆表面 | Clean | 青色方块 🟨 |
| **热处理炉** | 高温退火/氧化 | Furnace | 橙色方块 🟧 |
| **CMP抛光** | 化学机械抛光 | CMP | 灰色方块 ⬜ |

### 检测设备

| 设备类型 | 作用 | FabSim表示 |
|----------|------|------------|
| **量测设备** | 测量膜厚/线宽 | 白色圆角方块 |
| **缺陷检测** | 扫描晶圆缺陷 | 黄色圆角方块 |

## 三、缓冲与传输

### 缓冲区（Buffer）

```
作用：临时存放等待加工的晶圆批次（Lot）

    设备A          缓冲区          设备B
   [🟦加工中]  ←→  [🟪排队中]  ←→  [🟦加工中]
                  
                  容量限制：最多放10个Lot
                  排队规则：FIFO（先进先出）
```

**为什么需要缓冲？**
- 设备速度不匹配时暂存
- 故障时不会立刻停线
- 不同批次之间的衔接

### 传输路径

```
方式1：人工搬运（小厂/老厂）
方式2：天车系统（OHT）← 大厂主流
方式3：AGV小车

FabSim简化：就用连线表示，标注运输时间

    [设备A]══════[设备B]
         5分钟
```

## 四、一条完整产线示例

```
晶圆入口
    ↓
[清洗] → [氧化] → [光刻] → [刻蚀] → [沉积] → [检测] → 晶圆出口
            ↑________↓________↓________↓
              （缓冲区，每步之间都有暂存）

用FabSim的2D表示：

[🟨]→[🟧]→[🟦]→[🟥]→[🟩]→[⬜]→
   ↓    ↓    ↓    ↓    ↓
  [🟪]  [🟪]  [🟪]  [🟪]  [🟪]

数字 = 当前排队数量
```

## 六、概念解释

### Lot（批次）
- 一组晶圆，通常25片/批
- FabSim里用一个数字或圆点表示
- 在设备上加工、在缓冲中排队

### Recipe（工艺配方）
- 定义一个产品要经历哪些工序
- 比如：清洗(10min) → 光刻(30min) → 刻蚀(20min)
- FabSim里用表格配置

### WIP（在制品）
- 正在产线上流转的所有Lot
- FabSim实时统计显示

## 七、你现在要做什么

1. **先选3-4种设备**练手，比如：
   - 清洗 → 光刻 → 刻蚀 → 检测

2. **画一条简单产线**：
   ```
   [清洗] → [光刻] → [刻蚀] → [检测]
      ↓       ↓        ↓
     [暂存]  [暂存]   [暂存]
   ```

3. **配置一个Recipe**：
   - 产品A：清洗(10min) → 光刻(30min) → 刻蚀(20min) → 检测(5min)

4. **跑仿真**：
   - 扔10个Lot进去，看它们怎么流动
