# 半导体前道工艺说明

## 一、真实前道工艺：不止3个，循环N次

### 核心循环（反复进行）

```
真实芯片制造 = 这个循环重复 20~100+ 次

        第1层                第2层                第N层
    ┌─────────┐         ┌─────────┐         ┌─────────┐
    │薄膜沉积 │    →    │薄膜沉积 │    →    │薄膜沉积 │
    │  (CVD)  │         │  (CVD)  │         │  (CVD)  │
    └────┬────┘         └────┬────┘         └────┬────┘
         ↓                   ↓                   ↓
    ┌─────────┐         ┌─────────┐         ┌─────────┐
    │  光刻   │    →    │  光刻   │    →    │  光刻   │
    │ (Litho) │         │ (Litho) │         │ (Litho) │
    └────┬────┘         └────┬────┘         └────┬────┘
         ↓                   ↓                   ↓
    ┌─────────┐         ┌─────────┐         ┌─────────┐
    │  刻蚀   │    →    │  刻蚀   │    →    │  刻蚀   │
    │ (Etch)  │         │ (Etch)  │         │ (Etch)  │
    └────┬────┘         └────┬────┘         └────┬────┘
         ↓                   ↓                   ↓
    ┌─────────┐         ┌─────────┐         ┌─────────┐
    │  清洗   │         │  清洗   │         │  清洗   │
    │ (Clean) │         │ (Clean) │         │ (Clean) │
    └─────────┘         └─────────┘         └─────────┘
```

**为什么需要这么多层？**
- 现代芯片有几十层电路（金属层+绝缘层）
- 每一层都需要：沉积→光刻→刻蚀→清洗
- 先进制程（7nm/5nm）可能有100+层

## 二、完整前道工艺流程

```
晶圆进厂
    ↓
┌──────────────────────────────────────────────────────┐
│  1. 清洗 (Wet Clean)                                  │
│     - 去除杂质、颗粒                                  │
│     - 每次加工前后都要做                              │
├──────────────────────────────────────────────────────┤
│  2. 热氧化/沉积 (Oxidation/CVD/PVD/ALD)              │
│     - 生成氧化层或沉积薄膜                            │
│     - 可能是绝缘层、导电层、阻挡层                    │
├──────────────────────────────────────────────────────┤
│  3. 涂胶 (Coat)                                      │
│     - 晶圆表面涂光刻胶                                │
│     - 软烤固化                                        │
├──────────────────────────────────────────────────────┤
│  4. 光刻 (Lithography) ★核心步骤                      │
│     - 曝光：紫外线透过掩模版照射                      │
│     - 显影：溶解曝光/未曝光区域                       │
│     - 硬烤                                            │
├──────────────────────────────────────────────────────┤
│  5. 刻蚀 (Etch) ★核心步骤                             │
│     - 干法刻蚀（等离子体）或湿法刻蚀                  │
│     - 按照光刻胶图案去除材料                          │
│     - 去胶                                            │
├──────────────────────────────────────────────────────┤
│  6. 离子注入 (Ion Implantation) - 选择性             │
│     - 改变硅的电导率                                  │
│     - 形成晶体管源/漏极                               │
│     - 退火激活                                        │
├──────────────────────────────────────────────────────┤
│  7. 清洗 (Clean)                                     │
│     - 去除刻蚀残留                                    │
├──────────────────────────────────────────────────────┤
│  8. 量测/检测 (Metrology/Inspection)                 │
│     - CD-SEM：测量线宽                                │
│     - 缺陷扫描                                        │
│     - 膜厚测量                                        │
├──────────────────────────────────────────────────────┤
│  9. CMP抛光 - 选择性                                 │
│     - 化学机械抛光，平整化表面                        │
│     - 铜互连工艺中必需                                │
└──────────────────────────────────────────────────────┘
    ↓
循环回到步骤2，开始下一层...
    ↓
所有层完成 → 晶圆出厂 → 切割 → 封装
```

## 三、FabSim MVP 如何处理多层循环

### 方案1：单层验证（推荐MVP用）

```
只模拟一次核心循环，验证系统能跑通

[清洗] → [沉积] → [涂胶] → [光刻] → [刻蚀] → [检测]

配置一个Lot的工艺流程（单层）
等系统稳定后，再扩展多层
```

### 方案2：多层配置（Phase 2）

```
Recipe可以定义多层循环：

Layer 1: 沉积(30min) → 光刻(45min) → 刻蚀(30min) → 清洗(10min)
Layer 2: 沉积(30min) → 光刻(45min) → 刻蚀(30min) → 清洗(10min)
Layer 3: 沉积(30min) → 光刻(45min) → 刻蚀(30min) → 清洗(10min)
...
Layer 20: ...

同一个Lot在系统中循环20次，每次去同样的设备但加工不同层
```

### 方案3：简化为"等效层数"

```
如果真实要跑40层，仿真里可以先跑5层
按比例推算总时间和瓶颈
节省时间，验证逻辑
```

## 四、MVP建议：先做单层，但预留多层接口

```gdscript
# Recipe 数据结构设计
{
    "product_name": "8寸晶圆A",
    "total_layers": 1,        # MVP先设1，后期改20/40/100
    "base_process": [          # 基础工艺流程
        {"step": 1, "process": "clean", "time": 10},
        {"step": 2, "process": "deposit", "time": 30},
        {"step": 3, "process": "litho", "time": 45},
        {"step": 4, "process": "etch", "time": 30},
        {"step": 5, "process": "clean", "time": 10},
        {"step": 6, "process": "inspect", "time": 15}
    ]
}

# 运行时的逻辑
func process_lot(lot):
    for layer in range(1, recipe.total_layers + 1):
        lot.current_layer = layer
        for step in recipe.base_process:
            execute_step(lot, step)
```

## 五、常见误区澄清

| 误区 | 真相 |
|------|------|
| "前道就3个步骤" | ❌ 实际是9+个步骤，循环几十次 |
| "光刻只做一次" | ❌ 每层都要光刻，做几十次 |
| "仿真要跑完全部层数" | ❌ 可以用等效层数，或先跑单层验证 |
| "所有层用同样时间" | ⚠️ 不同层可能工艺不同，但MVP可以先一样 |

## 六、给你的建议

**MVP阶段（现在）：**
1. 配置一条单层工艺流程（6-7个步骤）
2. 验证Lot能从头到尾跑完
3. 测试设备排队、缓冲溢出等场景

**Phase 2（1个月后）：**
1. 把 `total_layers` 改成10或20
2. 看Lot在系统中循环多次的表现
3. 观察瓶颈工位是否会随层数变化

**Phase 3（对接MES后）：**
1. 导入真实的40层Recipe
2. 对比仿真预测 vs 真实生产数据
